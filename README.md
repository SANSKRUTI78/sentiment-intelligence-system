# Real-Time Sentiment Analysis Tool

## 📌 Overview

This project is a Python-based Real-Time Sentiment Analysis Tool that analyzes public opinion from text data such as social media posts, reviews, or comments.

It classifies text into:

* Positive 😊
* Neutral 😐
* Negative 😡

---

## 🚀 Features

* Real-time sentiment classification
* Text preprocessing pipeline
* Data visualization using bar charts
* Interactive web app using Streamlit
* Modular and scalable code structure

---

## 🛠️ Tech Stack

* Python
* NLTK (VADER Sentiment Analysis)
* Pandas
* Matplotlib
* Streamlit

## ⚙️ Installation

```bash
git clone https://github.com/your-username/sentiment-analysis-tool.git
cd sentiment-analysis-tool
pip install -r requirements.txt
```

---

## ▶️ Usage

Run the main script:

```bash
python main.py
```

Run the web app:

```bash
streamlit run app/streamlit_app.py
```
